# Подсказки по командной строке

Команда смены директорий
```sh
cd c:/folder_name
```

Команда отображения текущей директории. Для MacOs, Linux
```sh
pwd
```

Листинг текущей дие=ректории
Windows:
```sh
dir
```
Linux, MacOs:
```sh
ls
```

Удаление файла в Windows:
```sh
del <filename>
```
в linux, MacOs:
```sh
rm <filename>
```
